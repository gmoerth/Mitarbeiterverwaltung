Mitarbeiterverwaltung

Ein Unternehmen benötigt ein Programm zur Verwaltung von Mitarbeitern.
Das Unternehmen beschäftigt unterschiedliche Mitarbeitertypen. Alle Mitarbeiter (auch normale Mitarbeiter) verfügen über folgende Informationen 

- Fortlaufende eindeutige Mitarbeiter-ID
- Name
- Geburtsdatum
- Eintrittsdatum
- Grundgehalt (wird 14x pro Jahr ausgezahlt)

Für weitere Mitarbeitertypen gelten folgende Besonderheiten:

- Manager erhalten jährlich einen Bonus, welcher bei der Gehaltserhöhung um denselben Prozentsatz erhöht wird.
- Experten verfügen über ein Fachgebiet (string) und erhalten das Gehalt 15x jährlich
- (für "normale" Mitarbeiter gelten keine Besonderheiten)

Folgende Funktionen sollen für alle Mitarbeiter zur Verfügung stehen: 

- Jahresgehalt: das Jahresgehalt wird berechnet
- Gehaltserhöhung: das Grundgehalt wird um einen Prozentsatz erhöht
- Anstellungsdauer: die Dauer des Anstellungs-Verhältnisses wird berechnet
- Mitarbeiterblatt: alle Informationen werden ausgegeben (Konsole)

Für die Verwaltung der Mitarbeiter soll eine eigene Verwaltungsklasse implementiert werden, welche alle Mitarbeiter
(Mitarbeiter, Manager, Experten) in einer einzigen Liste führt. Die Klasse soll folgende Funktionalität zur Verfügung stellen:

- Hinzufügen eines Mitarbeiters / Managers / Experten
- Abrufen eines Mitarbeiters per Mitarbeiter-Nummer
- Gehaltserhöhung für alle Mitarbeiter durchführen
- Gehaltserhöhung für 1 Mitarbeiter durchführen
- Ausgabe eines Mitarbeiters / aller Mitarbeiter

Erstelle ein Klassendiagramm für die erforderlichen Klassen
